## 📌 Overview
This project proposes an adaptive traffic signal control system using **Deep Reinforcement Learning**.

A centralized **Dueling Double DQN (D3QN)** agent with **Prioritized Experience Replay** learns optimal traffic light policies across multiple intersections using SUMO simulation.

---

## 🧠 Key Features
- Reinforcement Learning based signal control
- Dueling Double DQN architecture
- Prioritized Experience Replay
- Parameter sharing across 88 intersections
- SUMO + TraCI integration
- Pure NumPy implementation

---

## ⚙️ Technologies
- Python
- NumPy
- Reinforcement Learning
- SUMO Simulator
